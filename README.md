# E-Commerce Sales & Revenue Analysis Dashboard

## Overview
This project analyzes e-commerce sales data using Power BI.

## Features
- Total Sales KPI
- Total Profit KPI
- Revenue Trend Analysis
- Profit Trend Analysis
- Category Analysis
- Region Analysis
- Top Products Analysis
- Interactive Slicers

## Tools Used
- Power BI
- Excel/CSV
- DAX
- GitHub

## Business Insights
- Identified top-performing products
- Compared regional sales performance
- Analyzed category-wise revenue contribution
- Tracked sales and profit trends
